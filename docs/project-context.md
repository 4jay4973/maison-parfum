# Maison Parfum

Luxury perfume ecommerce website.

Primary goal:
Demonstrate premium branding and user experience.

Secondary goal:
Demonstrate automation workflow from design to production-ready frontend.

The project prioritizes:
1. Visual quality
2. Maintainability
3. Performance
4. Feature completeness

Tech Stack:
- Next.js
- TypeScript
- Tailwind
- WooCommerce (future)

Design Goals:
- Premium
- Elegant
- Fashion-inspired

Existing Components:
- Button
- Container
- SectionTitle
- ProductCard
- Navbar

Always reuse existing components.
